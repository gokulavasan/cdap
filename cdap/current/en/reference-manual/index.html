CDAP Reference Manual: APIs, Licenses, and Dependencies¶

  • HTTP RESTful API: CDAP has an HTTP interface for a multitude of purposes: everything from sending data events to a Stream or to inspect the contents of a Stream through checking the status of various System and Custom CDAP services.
  • Javadocs: Javadocs of the Java APIs for writing CDAP Applications.
  • Java Client API: provides methods for interacting with CDAP from external Java applications.
  • Command-Line Interface API: methods for interacting with a CDAP instance from within a shell.
  • Licenses and Dependencies: License information for the CDAP Product and Documentation and CDAP Dependent Packages, Licenses, and License URLs.
